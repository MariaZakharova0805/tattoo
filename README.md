# Destin Tattoo
Tattoo and piercing website

![Project illustartion](https://github.com/MariaZakharova0805/tattoo/raw/main/screenshot.png)

### Project Features
- SPA
- Animation
- Picture slider
- Sending a pre-appointment details by mail from the site

## Deploy
<a href="https://tattoo-ten.vercel.app/" target="_blank">destin-tattoo.com</a>

## Technologies
- React
- TS
- JS
- CSS

## Development
NodeJS v8+ is required to install and run the project.

### Project setup
```sh
$ npm i
$ npm run dev
$ npm run build
```
